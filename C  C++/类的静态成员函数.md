### 静态成员函数
- 静态成员函数属于整个类所有
- 可以通过**类名或对象名**访问公有的静态成员函数

##### 定义：static修饰	


```cpp
#include <stdio.h>

class Demo
{
private:
    int i;
public:
    int getI();
    static void StaticFunc(const char* s);
    static void StaticSetI(Demo& d, int v);
};

int Demo::getI()
{
    return i;
}

void Demo::StaticFunc(const char* s)
{
    printf("StaticFunc: %s\n", s);
}

void Demo::StaticSetI(Demo& d, int v)
{
    //i = v;//静态成员函数不能直接访问成员变量的
	d.i = v;
}

int main()
{
    Demo::StaticFunc("main Begin...");
    
    Demo d;
	
	Demo::StaticSetI(d, 10);
	
	printf("%d\n", d.getI());
	
	Demo::StaticFunc("main End...");
    
    return 0;
}

```

#### 静态成员函数与普通成员函数
|  | 静态成员函数 | 普通成员函数 |
|--|--|--|
|所有对象共享  | yes | yes |
| 隐含this指针 | no | yes |
| 访问普通成员变量 | NO | yes |
| 访问静态成员变量 | yes | yes |
| 通过类名直接调用 | yes | no |
| 通过对象名直接调用 | yes | yes |

```cpp
#include <stdio.h>

//int gCount = 0;

class Test
{
private:
//public:
	static int mCount;
public:
	Test()
	{
		mCount++;
		//gCount++;
	}
	~Test()
	{
		mCount--;
		//gCount--;
	}
	static int getCount()
	{
		return mCount;
	}
	
};


int Test::mCount = 0; 


int main()
{
	printf("Count = %d\n",Test::getCount());//类名访问，公有静态成员变量
	
	Test t1;
	Test t2;
	
	printf("Count = %d\n",Test::getCount());//每个对象都有自己的成员变量
	printf("Count = %d\n",t1.getCount());
	printf("Count = %d\n",t2.getCount());
	
	Test * pt = new Test();//堆对象
	
	printf("count = %d\n", pt->getCount());//4
	
	delete pt;
	printf("Count = %d\n",Test::getCount());//每个对象都有自己的成员变量//3
	
	return 0;
}
```

### 小结：
- 静态成员函数没有隐藏的this参数，所以不能访问普通成员变量（函数）
- 静态成员函数通过**类名**或对象直接访问
- 静态成员函数只能直接访问静态成员变量（函数）

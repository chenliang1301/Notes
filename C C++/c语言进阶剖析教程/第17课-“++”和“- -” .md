17-2.c

```c
#include<stdio.h>

int main(){

	int i = 0;
	int j = 0;
	int k = 0;
	int s = 0;
	int a=0,b=0,c=0;

	s = ++i + ++j + ++k;//i=1,j=1,k=1,s=3
	c = a++ + b;//a=1,b=0,c=1?? 先运算，后自增a=0,c=0

	printf("i=%d\tj=%d\tk=%d\ts=%d\n",i,j,k,s);
	printf("a=%d\tb=%d\tc=%d\n",a,b,c);
	
	return 0;
}
```

1、C贪心，读取跟多字符再处理

2、用空格表示一个完整符号的休止符，开始处理

3、a++ 运算，先取值运算，后自增

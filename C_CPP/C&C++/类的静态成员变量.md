# 任意时候找当前类的对象个数

- 安全性不用全局变量
- 随时获取当前对象的个数

```cpp

#include <stdio.h>

//int gCount = 0;

class Test
{
private:
    int mCount;
public:
    Test():mCount(0)
    {
        mCount++;
        //gCount++;
    }
    ~Test()
    {
        mCount--;
        //gCount--;
    }
    int getCount()
    {
        return gCount;
    }
};

Test gTest;

int main()
{

    Test t1;
    Test t2;

    printf("Count = %d\n",gTest.getCount());//每个对象都有自己的成员变量
    printf("Count = %d\n",t1.getCount());
    printf("Count = %d\n",t2.getCount());
    return 0;
}

```

#### 静态成员变量的特性

- 在定义时用static
- 静态成员变量需要在类外单独分配空间
- 静态成员变量在程序累不位于全局数据区

#### 语法规则

- Type ClassName::VarName = value;

```cpp

#include <stdio.h>

//int gCount = 0;

class Test
{
private:
    static int mCount;
public:
    Test()
    {
        mCount++;
        //gCount++;
    }
    ~Test()
    {
        mCount--;
        //gCount--;
    }
    int getCount()
    {
        return mCount;
    }
};

int Test::mCount = 0;

Test gTest;

int main()
{

    Test t1;
    Test t2;

    printf("Count = %d\n",gTest.getCount());//每个对象都有自己的成员变量
    printf("Count = %d\n",t1.getCount());
    printf("Count = %d\n",t2.getCount());

    Test * pt = new Test();//堆对象

    printf("count = %d\n", pt->getCount());//4

    delete pt;
    printf("Count = %d\n",gTest.getCount());//每个对象都有自己的成员变量//3

    return 0;
}

```

#### 小结

- 类可通过static关键字定义静态成员变量
- 静态成员变量隶属类所有
- 每一个对象都可访问静态成员变量
- 静态成员变量存储在全局数据区
- 静态成员变量的生命周期为程序运行周期

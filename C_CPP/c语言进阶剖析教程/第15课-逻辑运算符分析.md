# 第15课-逻辑运算符分析

15-1.c

```c
#include<stdio.h>

int main(){

	int i = 0;
	int j = 0;
	int k = 0;
	++i || ++j && ++k;
	printf("%d\n%d\n%d\n",i,j,k);
	return 0;
}
```

1、优先级：&&高于||

2、短路：

       1）&&遇到真停止执行此条语句其他运算
    
       2）||遇到假则停止执行此条语句其他计算

3、++i || ++j && ++k;

      等价替换：      (true&&++i) || ++j && ++k;  

实际运算：      (true&&==i)

4、运算规则：从左向右

15-2.c

```c
#include<stdio.h>

int g = 0;

int fun(){

	printf("in fun...\n");
	return g++;
}

int main(){

	if(g||(fun()&&g)){//(true&&g)||(fun()&&(++g))
		printf("in main...\n");
	}

	printf("%d\n",g);	
	return 0;
}

/*
in fun...

1

*/
```

14-1.c

```c

int main(){

	char* p1 = 1;
	char* p2 = '1';
	char* p3 = "1";

	//printf("%s,%s,%s",p1,p2,p3);

```

2、1：4byte

      ‘a’：1byte    'a'+1 =='b'

      "a"：2byte   ‘a’ ‘\0’

14-2.c

```c
#include<stdio.h>
int main(){

	char c = ' ';
	while((' '==c)||('\t'==c)||('\n'==c)){
		scanf("%c",&c);
	}
	return 0;
}
```

1、单引号括起来，表示整数 如：‘a’

2、双引号括起来，表示指向字符串的指针 “abc”
